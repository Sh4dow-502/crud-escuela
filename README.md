# CRUD Escuela

Aplicación CRUD de alumnos desarrollada con **Frontend y Backend separados**.

## Tecnologías usadas

### Frontend

- HTML5
- JavaScript
- Bootstrap 5

### Backend

- Node.js
- Express
- PostgreSQL

---

## Funcionalidades

- Crear alumnos
- Listar alumnos
- Editar alumnos
- Eliminar alumnos
- Validación de teléfono
